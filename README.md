# Coding challenge - Number Converter

**Tech stack:**

- Java
- Spring Boot
- Maven
- Lombok - boilerplate code reduction
- MySQL
- Docker Compose

**Tools:**

- Postman - collection file: NumberConverterApiRequestsCollection.postman_collection.json

**How to build:**

Execute the shell script:

```
./backend-build.sh
```

**How to start backend app:**

Execute the shell script:

```
./backend-start.sh
```
